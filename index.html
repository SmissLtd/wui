WEB-usage inspector(WUI) installation

  • Download and install the plugin
  • Go to Firefox add-on manager
  • Go to add-ons
  • Select WEB-usage inspector add-on
  • Click on settings
  • Specify computer name and server url
  • To start experiment, click on "Start" button on Firefox toolbar
  • Specifiy user name in popup and click OK
  • To stop experiment, click on "Stop" button on Firefox toolbar

Requirements

  • Firefox 38+
